# User Access Review
User access to systems and data is periodically reviewed to ensure access remains appropriate and aligned with each individual's role. User access reviews reduce the risk of unauthorised or excessive access by confirming that only authorised individuals retain access and that access is revoked when no longer required.

## Control Attributes
* Access reviews are performed on a periodic basis (e.g. quarterly)
* Access is reviewed and approved by an appropriate system or data owner
* Inappropriate or excessive access identified during the review is remediated in a timely manner
